# Hangman

## Overview

Hangman is a web project inspired by the classic word-guessing game. This project challenges players to guess a hidden word by selecting letters. Each incorrect guess results in the progression of a hangman drawing. The game ends when the hangman drawing is complete. The game interface displays the game title at the top left and provides clues to help players guess the word.

## Installation

To run this project locally, follow these steps:

1. Clone the repository.
2. Open the `index.html` file in your preferred web browser.

## Features

- Game Title Display: The game title is shown at the top left corner of the interface.
- Word Clues: Clues to help players guess the hidden word are displayed at the top right.
- Interactive Gameplay: Select letters to guess the word and avoid completing the hangman drawing.
- Loss Condition: The game ends if the hangman drawing is completed due to incorrect guesses.

## Technologies Used

- HTML
- CSS
- JavaScript

## Gameplay

- Guess the hidden word by selecting letters to avoid completing the hangman drawing.
- Each incorrect guess progresses the hangman drawing.
- The game ends if the hangman drawing is completed.
